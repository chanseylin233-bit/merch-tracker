# 周边拼团记录工具开发完成

**时间**: 2026-04-29
**状态**: ✅ 完成

## 项目概述

移动端优先的 React Web 应用，用于记录苏茜及周边拼团订单。

## 技术栈

- React 19 + TypeScript
- Vite
- React Router DOM
- localStorage（无后端）

## 项目位置

```
E:\merch-tracker\
```

## 核心功能

### 1. 订单管理
- 新增/编辑订单
- 删除订单
- 状态流转（16种状态）
- 付款记录、退款记录、进度时间线

### 2. 页面功能
- **首页**：待办提醒（待付款、待排发、即将截止、长期未更新）
- **列表**：筛选（状态/类型/群/团长）、搜索
- **详情**：完整信息、进度时间线、操作按钮
- **统计**：年度汇总、按类型/角色分布
- **设置**：备份导出、数据导入、清空数据

## 运行方式

```powershell
cd "E:\merch-tracker"
npm run dev
```

访问 http://localhost:5173

## 状态流转

观望中 → 已报名 → 拼团中 → 拼团成功待付款 → 已付定金待尾款 → 待补尾款 → 已付款待发货 → 已到团长处待排发 → 已申请排发待发货 → 已发货待收货 → 已收货已完成

终止状态：未拼成、流团待退款、已退款、部分退款、异常待确认、已放弃/已转单

## 结论

项目已完成核心功能开发，可正常运行。数据存储在浏览器 localStorage，用户需定期备份。
